Registers:

  • A (16 bits) - Accumulator
    1. H (High Part)
    2. D (Low Part)
  • B (16 bits) - Buffer
  • X (16 bits) - X Index
  • Y (16 bits) - Y Index
  • I (16 bits) - Instruction Pointer
  • S (12 bits) - Stack Pointer
  • P (8 bits) - Processor Status

Processor Status:

Bits: NV--FIZC

  • N - Negative
  • V - Overflow
  • F - Float Mode
  • I - Interrupt Disable
  • Z - Zero
  • C - Carry

Addressing Modes:

  • Implied (IMP)
  • Accumulator (A)
  • Data Buffer (B)
  • Constant Byte (IMM)
  • Constant Word (DIM)
  • Relative (REL)
  • Direct (DIR)
  • Direct + X (DIR, X)
  • Direct + Y (DIR, Y)
  • Zero Page (ZPG)
  • Zero Page + X (ZPG, X)
  • Zero Page + Y (ZPG, Y)
  • Indirect (X)
